# 素材清单

## 1. PPT素材

| 素材 | 用途 | 建议风格 | 来源建议 |
|---|---|---|---|
| 米白纸纹背景 | PPT内页背景 | 干净、低对比、像书页 | 自制纹理、可商用素材库 |
| 红色渐变封面 | 第1页封面 | 深红到暖红，庄重不刺眼 | PPT自制渐变 |
| 书本或文献图标 | 封面、经典导读页 | 线性图标，颜色统一 | Iconfont、可商用图标库 |
| 时间轴图标 | 第3页理论脉络 | 简洁节点、箭头 | PPT形状自制 |
| “守正/创新”关系图 | 第4页 | 左右对照、中间交汇 | PPT形状自制 |
| 中国地图或山河轮廓 | 第5页、第12页 | 淡化背景，不喧宾夺主 | 可商用矢量素材 |
| 传统文化纹样 | 第6页 | 淡金色或浅棕色，透明度低 | 自制纹样、可商用素材 |
| 科技/乡村/生态图标 | 第7页现实案例 | 统一线性风格 | 可商用图标库 |
| WebDemo二维码 | 第9页 | 课堂扫码访问 | 本地运行后用二维码工具生成 |
| 小组照片3张 | 第8页、活动总结 | 真实自然、清晰明亮 | 小组自摄 |

### 小组照片建议

1. 小组成员围坐阅读文献，桌面有笔记、荧光笔和教材。
2. 小组讨论PPT结构或在白板上梳理“两个结合”。
3. 成员共同测试WebDemo或修改AI视频脚本。

## 2. WebDemo素材

| 素材 | 当前项目文件 | 用途 | 替换说明 |
|---|---|---|---|
| 首页背景图 | `WebDemo/public/reading-hero.png` | 首页Hero背景 | 可替换为校园阅读或书页主题横图 |

| 小组照片占位2 | `WebDemo/public/group-photo-2.svg` | 交流讨论照片 | 替换为真实照片，文件名可保持不变 |

### WebDemo素材风格

- 图片比例优先选择16:9或4:3，避免过窄竖图影响投屏。
- 颜色建议与页面保持一致：深红、米白、暖金、深棕、自然绿色。
- 不使用娱乐化表情包、夸张贴纸或与课程无关的装饰图。

## 4. 版权与规范注意事项

1. 优先使用小组自摄照片、校园实拍素材和可商用素材。
2. 图片和视频不要带水印，不使用来源不明的影视片段。
3. 人物照片需征得成员同意，避免拍到无关同学的清晰正脸。
4. AI生成素材要避免出现错误文字、变形书页、夸张宣传画风。
5. 所有素材服务于经典品读主题，不为装饰而堆砌。
6. 最终提交前统一压缩图片大小，保证PPT和WebDemo打开流畅。

## 1. 主题定位

核心问题：为什么马克思主义必须中国化时代化？作为新时代大学生，我们如何把经典阅读转化为理解中国、观察时代、规划人生的思想能力？

逻辑主线：

1. 从经典出发：读什么，为什么读。
2. 从毛概进入：马克思主义中国化时代化的历史脉络。
3. 从文本展开：守正创新、“两个结合”、问题导向。
4. 从现实验证：中国式现代化、科技自立自强、乡村振兴、生态文明。
5. 从青年落脚：形成方法、坚定方向、提升本领。

## 3. WebDemo最终要求

WebDemo不做普通网页，也不做杂乱模块堆叠。它必须是课堂投屏用自动翻书总结演示。

- 使用Vite + React + TypeScript。
- 适配GitHub Pages子路径 `/mg/`。
- 翻页动画使用本地化官方 `turn.js` 插件驱动，按官方 magazine 示例保持轻量页面结构，保留插件原生折页、阴影、渐变和手动拖拽能力。
- 深红动态视频背景，庄重、热烈、有课堂感染力。
- 主体为左右书页、书脊、纸张厚度、翻页阴影、背页纹理。
- 标题使用宋体/中宋类字体栈，正文使用清晰的中文阅读字体栈。
- 图片真实、清晰、与章节内容直接相关。
- 样式不得覆盖 `turn.js` 生成的 `turn-page-wrapper`、临时折页层和渐变层；按钮、键盘、自动播放统一调用插件原生 `next`、`previous`、`page` 方法。

自动演示章节：

1. 开卷：从一个问题翻开经典，对应PPT 1-2。
2. 主线：理论在回答中国问题中生长，对应PPT 3。
3. 方法：守正创新不是两选一，对应PPT 4。
4. 根脉：两个结合让理论扎根中国，对应PPT 5-6。
5. 现实：问题导向让理论进入现场，对应PPT 7。
6. 共读：把讨论留在真实现场，对应PPT 8。
7. 展厅：数字展厅服务思想表达，对应PPT 9-10。
8. 青年：从理论自觉到实践担当，对应PPT 11。
9. 合卷：用三句话收束整套汇报，对应PPT 12。

线下讨论现场影像：

- `WebDemo/public/photos/discussion-reading.jpg`：读原文。
- `WebDemo/public/photos/discussion-notes.jpg`：辨问题。
- `WebDemo/public/photos/discussion-review.jpg`：定表达。

## 4. 自检与部署流程

1. 检查Markdown、PPTX、DOCX、WebDemo逻辑是否一致。
2. `cd WebDemo && npm run build`。
3. 启动本地预览，截图检查桌面和移动端。
4. 扫描WebDemo可见文本，确保没有制作说明感文字。
5. 提交并推送到 `https://github.com/2711944586/mg` 的 `main` 分支。
6. 通过GitHub API检查GitHub Actions部署结果。

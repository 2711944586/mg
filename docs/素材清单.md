# 素材清单

## 1. PPT素材

| 素材 | 用途 | 建议风格 | 来源建议 |
|---|---|---|---|
| 深红动态感背景 | 封面、章节页、结尾 | 庄重、热烈、有纸纹或绸面流动感 | PPT自制渐变、视频截图、可商用纹理 |
| 米白纸页纹理 | 内页底色 | 像书页，低对比，文字清晰 | 自制纹理、可商用纸纹 |
| 国家图书馆阅览室照片 | 第2页开卷、WebDemo开场 | 安静、有真实阅读氛围 | Wikimedia Commons |
| 古籍书页照片 | 第4页守正创新、第6页文化根脉 | 有书卷质感，避免过度古风 | Wikimedia Commons |
| 乡村与传统建筑照片 | 第5-6页两个结合 | 真实、明亮、能体现中国实际和文化根脉 | Wikimedia Commons 或小组自摄 |
| FAST/高铁/生态/乡村照片 | 第7页现实案例 | 清晰、有冲击力，可投屏放大 | Wikimedia Commons |
| WebDemo截图或二维码 | 第9页展示入口 | 直接展示自动翻书主画面 | 部署页面截图和二维码工具 |
| 小组线下讨论照片3张 | 第8页共读过程、活动总结 | 自然、明亮、桌面有文献/笔记/电脑 | 小组自摄 |

### 小组照片建议

1. 读原文：小组成员围坐阅读文献，桌面有教材、笔记和划线痕迹。
2. 辨问题：围绕“两个结合”“守正创新”讨论结构，可拍到纸面提纲或白板。
3. 定表达：共同检查PPT、WebDemo或主题短片，画面要真实自然。

### WebDemo素材风格

- 只使用能讲清内容的图片，不为装饰堆图。
- 主图优先横图或可横向裁切，避免低清竖图承担大面积画面。
- 图片来源统一记录在 `WebDemo/public/photos/CREDITS.md`。
- 页面逻辑为“开卷-主线-方法-根脉-现实-共读-展厅-青年-合卷”，素材必须服务这条线。

## 4. 版权与规范注意事项

1. 优先使用小组自摄照片、校园实拍素材和可追溯的公共授权素材。
2. 图片和视频不要带水印，不使用来源不明的影视片段。
3. 人物照片需征得成员同意，避免拍到无关同学的清晰正脸。
4. 自摄或授权画面要避免错误文字、变形书页、夸张宣传画风。
5. 所有素材服务于经典品读主题，不做无意义装饰。
6. 最终提交前统一压缩图片大小，保证PPT和WebDemo打开流畅。
